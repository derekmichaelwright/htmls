Introduction

R can be used as a caclulator

5 + 3 # = 8
## [1] 8
24 / (1 + 2) # = 24 / 3 = 8
## [1] 8
2^3 # = 2 * 2 * 2 = 8
## [1] 8
sqrt(64) # 8 * 8 = 64
## [1] 8
-log10(0.05 / 5000000) # = 8
## [1] 8

Information can be stored in user defined objects. e.g., xx

xx <- -log10(0.05 / 5000000)
xx
## [1] 8

R has many useful built in functions

Users can also create their own functions

customFunction <- function(x) {
  mean(x) + sd(x)
}
customFunction(1:10)
## [1] 8.52765

Plotting

plot(x = x3$y1, y = x3$y5)

plot(x = x3$y1, y = x3$y5, col = x3$y2)

plot(x = x3$y7, y = x3$y8, type = "both")
library(ggplot2)

mp <- ggplot(x3, aes(x = y7, y = y8))
mp + geom_point()

mp + geom_point(aes(color = y2))

mp + geom_line()

mp + geom_line(aes(color = y2))

mp + geom_smooth()

mp + geom_smooth(method = "loess")

mp + geom_smooth(method = "lm")

mp <- ggplot(x3, aes(x = y2, y = y7))
mp + geom_boxplot()

mp + geom_boxplot() + geom_point()

mp <- ggplot(x3, aes(x = y3))
mp + geom_histogram(stat = "count")
